# Burned Area Detector - QGIS Plugin

**Author:** Ghulam Abbas Zafari  
**Institution:** Politecnico di Milano  
**Degree:** MSc Geoinformatics Engineering

## Features
- Burned Area Detection with fuzzy logic
- Burn Severity Classification (USGS)
- Validation with Confusion Matrix
- SCL Cloud/Water Masking
- Change Detection Analysis
- Agreement Map Generation
- Batch Processing

## Installation
1. Open QGIS
2. Plugins → Manage and Install Plugins
3. Search "Burned Area Detector"
4. Click Install
